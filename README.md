# 这个b班要上吗？

一个交互式的工作机会评估工具，帮助求职者理性评估工作机会。

## 功能特点

- 🎯 **10维度评估系统**：全面评估工作机会的各个方面
- 💯 **智能评分**：自动计算总分，超过60分即为好公司
- 🎨 **现代化UI**：美观的渐变设计和流畅的动画效果
- 📱 **响应式设计**：完美适配桌面和移动设备
- 🔄 **实时反馈**：评分变化实时更新，结果即时显示
- 📤 **结果分享**：支持分享评估结果

## 评估维度

每个维度满分10分，总分100分：

1. **薪资待遇** - 月薪5,000以上且准时到账
2. **工作时间** - 双休 + 8小时工作制
3. **五险一金** - 齐全且合同手续合理
4. **带薪年假** - 有带薪年假福利
5. **准时下班** - 不加班，准时下班
6. **年终奖与薪酬透明** - 年终奖 + 薪酬制度透明
7. **通勤便利** - 路程30分钟以内
8. **工作氛围** - 领导不PUA，同事好相处
9. **成长空间** - 能学到东西，有上升空间
10. **公司前景** - 前景好，不担心失业

## 安装依赖

```bash
npm install
```

## 开发模式

```bash
npm run dev
```

这将启动开发服务器，自动在浏览器中打开 http://localhost:8888

## 生产模式

```bash
npm run build
npm start
```

构建后的文件将输出到 `dist` 目录，然后可以通过 `npm start` 预览。

## 使用方法

1. 启动开发服务器后，在浏览器中打开网站
2. 根据实际情况，为每个维度选择相应的分数（0分、5分、10分）
3. 系统会自动计算总分并显示评估结果
4. 超过60分即为好公司，值得考虑！
